LC521. Longest Uncommon Subsequence I

發表於 2019-02-25 | 分類於 String

Description

  • 子序列(subsequence)：一個序列刪除其中某些元素，但不改變其他的順序，稱為某序列的子序列。

  • 非共同子序列(uncommon subsequence)：給定兩個序列，若其中有個序列的子序列，不是另一個序列的子序列，就稱此子序列為非共同子序列。

  • 本題要求找出最長的非共同子序列的長度。若沒有任何非共同子序列，回傳-1。
